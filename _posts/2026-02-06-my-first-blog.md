## Abstract
To create a website for processing food orders for a cloud kitchen to also create an admin option to add new dishes to the kitchen.

## Introduction
The aim is to create a user-friendly website for ordering food online. 
The dishes available in this cloud kitchen will be nutritious and healthy. 
The objective is to provide a website for ordering tasty but healthy food.

## Methodology
Existing cloud kitchen websites shall be studied to get ideas regarding the best features to be made available in my website. 
Research and use the latest programming constructs in Vue.js and Express/Node.js in order to optimize the source code. 

## Discussion
This project will use latest web technologies to produce a commercial website for a cloud kitchen. 
The purpose of the project is to use the technologies learnt during the course of my BSc program. 
The source code shall be modular and easily maintainable.

## Conclusion
The purpose of this project is apply the Web Development technologies learnt during my BSc program and develop a professional looking website for ordering food online.
